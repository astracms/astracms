# Changelog

Stay up to date with the latest changes and improvements to AstraCMS.

## v1.0.0

Initial release of AstraCMS with:

- Core API client package
- Framework integrations for React, Next.js, Vue, Nuxt, Svelte, and Astro
- RESTful API with full CRUD operations
- TypeScript support with Zod schemas
